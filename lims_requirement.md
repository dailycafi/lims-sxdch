# LIMS需求

# 需求分析文档整理模版
 
# 权限
 
l  每个用户都用自己的用户名、密码，以进行系统登录
l  所有人员都：可以查看各个模块信息及统计信息，是流程审批的环节之一
l  计算机或系统管理员：拥有最大权限，可以给实验室人员分配权限
l  研究室主任：无额外权限
l  分析测试主管：无额外权限~[FW1]~ ~[o2]~ 
l  样本管理员：除权限设置或系统设置外的所有操作
l  项目负责人：申请领用，归还，转移，销毁，归档
l  分析测试员：申请领用，归还，转移，销毁，归档
l  QA：无额外权限
 
# 审计追踪
l  任何操作人员的任何操作，均需留痕
l  信息包括：日期，时间，操作内容，操作人，备注
l  记录条目按照时间顺序排序，不允许任何人删除，修改
 
# 电子签名
l  部分关键过程，关键内容的修改，删除等操作，需要操作人员再次通过身份认证

**(八)**   **统计查询：所有人**
l  存取记录（流水）：样本从接收到销毁的全部过程中，任何的操作过程、位置移动，都需要自动生成存取记录条目，默认按时间顺序排序，排序方式可选（如按受试者编号排序等）
l  每一条存取记录内容包括：日期，时间，样本编号，保存位置，检测类型，临床机构编号，受试者编号，采血序号，采血时间，周期/组别，正份/备份，用途，特殊事项，备注
l  按日期，时间查询：
l  输入日期时间，显示时间段内全部样本存取记录
l  按单个样本编号查询：
l  输入样本编号，显示该样本的所有存取记录，默认按时间排序。
l  自动计算该样本的存取次数（注：在备注中如有未检测等说明内容，则该次存取不计入暴露时间的计算）
l  自动计算该样本单次次存取的暴露时间（如存取5次，则自动计算每次存取的暴露时间）
l  自动计算该样本多次存取的总暴露时间（如存取5次，则自动计算2次，3次，4次，5次存取的总暴露时间）
l  按用途查询：
l  输入用途，显示此用途的全部样本存取记录
 
用途（会根据需求发生变化）
首次检测：临床样本的第一次检测
重测：部分临床样本第一次检测后，由于某种原因，进行的第二次，第三次检测
ISR：根据方案要求，部分临床样本进行重复检测，可能是第二次，也可能是第三次检测
 
例如：
l  A0123这个样本，首次检测后，未重测，也未在ISR中进行检测
